The webpage is built using html css javascript

This webpage is built using html for scripting, css for styling and javascript for some scripting.
In html we used some simple commonly used elements,such as div,ul.li...
On the other hand, we used in styling media queries for a responsive design, simple bright colors for a clear and friendly interface(blue and white colors), used pseudo-classes such as :hover for some animation and a bit of transition...
Using javascript was for the hamburger menu to open and close using the event listener ("click"), and to hide it on certian screen width,or show also using the event listener ("resize").
